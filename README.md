# 🧠 MindGuard AI — Student Mental Health Burnout Prediction System

**MindGuard AI** is an intelligent, machine learning‑powered web application that predicts student burnout risk based on academic, mental health, and lifestyle factors. Built with **Streamlit** and a **Random Forest classifier**, it provides real‑time risk assessment, interactive analytics, and personalized recommendations.

## ✨ Features

- **AI‑Powered Prediction** – Uses a trained Random Forest model (9 input features) to classify burnout risk into **Low**, **Medium**, or **High**.
- **Interactive Dashboard** – Glass‑morphism UI with real‑time sliders and selectors.
- **Visual Analytics** – Dynamic Plotly charts:
  - Burnout Risk Gauge
  - Class Probability Bars
  - Student Radar Profile (normalized metrics)
  - Feature Importance Graph
- **Personalized Recommendations** – Actionable tips based on the user’s inputs (sleep, stress, study load, etc.).
- **Model Status Indicator** – Shows whether the ML model is loaded and ready.
- **Responsive Design** – Works on desktop, tablet, and mobile.

## 🛠️ Tech Stack

- **Frontend & Backend**: Streamlit (Python)
- **Machine Learning**: Scikit‑learn (Random Forest)
- **Data Processing**: Pandas, NumPy
- **Visualization**: Plotly
- **Styling**: Custom CSS (Glassmorphism, dark theme)
